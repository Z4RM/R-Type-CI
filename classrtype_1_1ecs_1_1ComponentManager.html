Adds a component to an entity.

If the component type does not yet exist, a new sparse set (SparseSet) is created for that type. The component is then added to the specified entity.

T * rtype::ecs::ComponentManager::getComponent ( unsigned int entity)
inline

Retrieves a component associated with an entity.

If a sparse set for the component type exists and the entity has a component of that type, it is returned. Otherwise, returns nullptr.

Template Parameters
TThe type of the component.
Parameters
entityThe unique identifier of the entity.
Returns
A pointer to the component, or nullptr if the entity does not have a component of this type.
