Detailed Description

Manages keyboard input actions for entities.

The InputHandler component allows associating keyboard keys with specific actions using a map. This enables easy handling of input-driven behavior in the RType game.

std::unordered_multimap<sf::Keyboard::Key, std::pair<sf::Event::EventType, std::function<void()> > > rtype::components::InputHandler::keyActions

A multimap associating keyboard keys with actions.

Each key in the map corresponds to an sf::Keyboard::Key, and each action is represented as a pair of sf::Event::EventType std::function<void()>. This allows flexible handling of key presses, binding specific functions to them.
