unsigned int rtype::ecs::EntityManager::createEntity ( )
inline

Creates a new entity.

If there are recycled IDs available, the oldest ID is reused. Otherwise, a new unique ID is generated. The created entity is added to the set of active entities.

Returns
The unique ID of the newly created entity.

◆ destroyEntity()

void rtype::ecs::EntityManager::destroyEntity ( unsigned int entity)
inline

Destroys an entity.

Marks the entity as inactive by removing it from the set of active entities and adds its ID to the queue of available IDs for future reuse.

Parameters
entityThe unique ID of the entity to destroy.

std::unordered_set< unsigned int > rtype::ecs::EntityManager::getEntities ( ) const
inline

Retrieves the set of active entities.

This method provides access to the set of active entities currently managed by the ECS framework. Each entity is represented by an unsigned integer.

Returns
A constant reference to an std::unordered_set containing the IDs of all active entities.
